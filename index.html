Documentation et Présentation d'équipe

Documentation

Bienvenue dans notre documentation. Vous trouverez ici toutes les informations nécessaires pour configurer, installer et utiliser notre projet.

Installation : Téléchargez le projet, ouvrez un terminal et exécutez npm install pour installer les dépendances.

Utilisation : Lancez npm start pour démarrer l’application en mode développement.
